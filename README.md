### 安装/卸载:
Centos:

```BASH
yum install wget -y &&
wget https://github.com/KLDGodY/mtproxy/raw/master/mtproxy.sh &&
bash mtproxy.sh
```

Ubuntu/Debian:

```BASH
apt-get install wget -y &&
wget https://github.com/KLDGodY/mtproxy/raw/master/mtproxy.sh &&
bash mtproxy.sh
```
